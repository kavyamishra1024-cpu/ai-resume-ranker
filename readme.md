# AI Resume Ranker

An AI-powered Resume Ranker built using Python and Streamlit.

## Features
- Upload PDF resumes
- Extracts text from resumes
- ATS-style resume analysis
- Simple and interactive UI

## Technologies Used
- Python
- Streamlit
- pdfplumber
- GitHub
